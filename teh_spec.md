## 🧱 Технологический стек

### Ключевой принцип: **Один файл — вся система**

| Компонент | Технология | Способ подключения |
|-----------|-----------|-------------------|
| **Разметка** | HTML5 | Встроенный |
| **Стилизация** | Tailwind CSS | CDN (`<script>`) |
| **Иконки** | Font Awesome | CDN (`<link>`) |
| **Шрифты** | Google Fonts | CDN (`<link>`) |
| **База данных** | SQLite через sql.js | CDN (`<script>`) |
| **Хранилище** | localStorage + IndexedDB | Встроенный JS |
| **Импорт CSV** | Встроенный JS | FileReader API |
| **Экспорт CSV** | Встроенный JS | Blob + download |
| **Бизнес-логика** | Vanilla JS | Встроенный |

## 💾 Стратегия хранения данных

### Трёхуровневая система сохранения

```
┌─────────────────────────────────────────────────────────────┐
│                    УРОВЕНЬ 1: localStorage                   │
│  • Быстрое сохранение текущей сессии                        │
│  • Мгновенный доступ к данным                               │
│  • Лимит: ~5-10 MB                                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                    УРОВЕНЬ 2: IndexedDB                      │
│  • Хранение больших объёмов данных                          │
│  • Асинхронная работа без блокировки UI                     │
│  • Лимит: до 50% свободного места на диске                  │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                 УРОВЕНЬ 3: Экспорт в .db файл                │
│  • Резервное копирование                                    │
│  • Перенос данных между устройствами                        │
│  • Архивирование истории                                    │
└─────────────────────────────────────────────────────────────┘
```

## 🔧 Ключевые модули системы

### 1. Модуль работы с данными
- **CRUD операции** — создание, чтение, обновление, удаление записей об ошибках
- **Фильтрация** — по ученику, типу ошибки, дате, предмету
- **Сортировка** — по частоте ошибок, по дате, по ученику

### 2. Модуль генерации фактов
- Автоматическое формирование текста для родителей
- Выбор тональности (строгий/мягкий/фактический)
- Копирование в буфер обмена одним кликом

### 3. Модуль импорта/экспорта
- **Импорт CSV** — загрузка данных из Яндекс.Учебника, Skysmart, Учи.ру
- **Экспорт CSV** — выгрузка для анализа или резервного копирования
- **Экспорт .db** — полное сохранение базы данных

### 4. Модуль визуализации
- Дашборд с блоком «Красный флаг» (до 5 фамилий)
- Детальная карточка ученика с историей ошибок
- Прогресс-трекинг (исчезнувшие ошибки)
